# Outdoor Event Service Enquiries Website

This project delivers a responsive website designed to streamline outdoor event service enquiries. It provides an intuitive user interface for both individuals and businesses to quickly submit detailed service requests via WhatsApp, specifically for crowd control fencing and stage setup.

## Features

*   **Responsive Design:** Adapts seamlessly to various screen sizes, from mobile devices to desktop computers.
*   **WhatsApp Enquiry Generation:** Automatically constructs a pre-filled WhatsApp message with enquiry details, ready for submission.
*   **Dual Enquiry Forms:** Separate forms cater to the specific needs of individual clients and business clients.
*   **Service Selection:** Clearly presents options for crowd control fencing and stage setup.
*   **Client-Side Validation:** Ensures required fields are completed before generating the WhatsApp message.

## Technologies Used

*   **HTML5:** For structuring the content of the web pages.
*   **CSS3:** For styling the user interface, ensuring a modern and visually appealing design.
*   **JavaScript:** For interactivity, form handling, and generating the dynamic WhatsApp enquiry links.

## Getting Started

To get a local copy up and running, follow these simple steps:

1.  **Clone the repository (or download the ZIP):**
    ```bash
    git clone <repository_url>
    ```
2.  **Navigate to the project directory:**
    ```bash
    cd Project-1
    ```
3.  **Open `index.html`:**
    Simply open the `index.html` file in your preferred web browser. No special server setup is required for basic functionality.

## Usage

1.  Fill out the enquiry form, selecting whether you are an individual or a business.
2.  Provide details about the event, including the desired services (crowd control fencing, stage setup).
3.  Click the "Submit Enquiry" or similar button.
4.  A WhatsApp chat window will open with a pre-populated message containing your enquiry details. Review and send the message.
