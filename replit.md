# Outdoor Event Services

## Overview
A static website for an outdoor event services company offering crowd control fencing and stage setup services. Features a contact form that generates WhatsApp messages for customer inquiries.

## Project Architecture
- **Type**: Static HTML/CSS/JavaScript website
- **No build process required**
- **No backend/database**

## Files
- `index.html` - Main page with hero section, services, and contact form
- `styles.css` - Styling with responsive design
- `script.js` - Form handling, live preview, and WhatsApp integration

## Development
- Served via Python's built-in HTTP server on port 5000
- Run command: `python3 -m http.server 5000 --bind 0.0.0.0`

## Deployment
- Static deployment from root directory (`.`)
